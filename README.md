# Quiz Site 🎓

**Quiz-Site** is a simple learning project designed for **10–11 grade students** in a Python programming course. Its purpose is to help beginners practice Python by exploring a basic quiz/test application and understanding how Python can be used to build interactive programs.

---

## 🧠 About the Project

This repository contains a **Python-based quiz application** that lets users take tests with predefined questions and interact with the program by choosing answers. The main goals are to:

- Practice fundamental Python concepts (loops, functions, data structures)
- Learn how to structure a simple project
- See how input/output works in a real-world-style application
- Get hands-on experience with running and modifying Python code

(*Note: This project does not depend on advanced frameworks, making it ideal for beginners.*)

---

## 🚀 Features

✔️ A collection of quiz questions  
✔️ Command-line based user interaction  
✔️ Correct/incorrect answer feedback  
✔️ Final score display  
✔️ Easy code structure for learning and extension
